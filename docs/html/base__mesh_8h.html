Detailed Description

This is the base classes of mesh, which describe ordered and indexed data sets. Depending on application, there are different data saved on the mesh. The intersection points of mesh lines are called grid points, the element enclosed by mesh lines (2D) or faces (3D) called cells. The mesh line or face are also called cell faces. Grid points are also called cell corners.
